# ipydoodle

![image](docs/images/ipydoodle.png)

ipydoodle은 학생들을 위한 2D 시각화 라이브러리입니다.

ipydoodle에서는 자유롭게 World를 만들고 Object를 만들고 움직일 수 있습니다.

이외에도 학생들이 만들고 싶은 모든 것들을 자유롭게 만들 수 있습니다.

여러분도 ipydoodle과 함께 자신만의 작품을 만들어보세요!!!
